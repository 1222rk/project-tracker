# 📋 Project Status Reporter

A simple web-based tool to generate executive project status reports.

It allows you to log milestones, risks, decisions, and next steps — then instantly preview and download a polished briefing document.

✨ Features-

Scenario RAG status tracking (Green / Amber / Red)

Real-time live preview

Milestone, risk, decision, and next step management

Inline editing and deletion

Auto-save to browser storage

Export report as a printable PDF

🧮 What It Tracks-

Overall project RAG status

Milestones with due dates and owners

Risks with impact, likelihood, and mitigation

Key decisions with owners and dates

Next steps with owners and due dates

Milestone status overview chart

🚀 Getting Started-

Run locally: Download or clone the repo Open index.html in your browser
